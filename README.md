# US Flight Performance Dashboard

## Project Overview
This project analyzes nearly **2 million commercial flight records from major US airports** to identify patterns in flight delays and cancellations. Using **Power BI**, the raw data was transformed into an interactive dashboard that enables stakeholders to explore flight performance and understand the factors contributing to operational disruptions.

## Objectives
- Analyze flight delays and cancellations across major US airports.
- Identify the primary causes of delays and cancellations.
- Compare airline performance based on operational metrics.
- Examine trends by airport and day of the week.
- Provide an interactive solution to support data-driven decision-making.

## Tools & Technologies
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel / CSV Dataset

## Dashboard Features
- Total Flights, Delayed Flights, and Cancelled Flights KPIs
- Airline-wise delay analysis
- Airport-wise performance insights
- Cancellation reason analysis
- Day-of-week trend analysis
- Interactive slicers and filters for dynamic exploration

## Key Insights
- Delay patterns differed significantly across airlines and airports.
- Certain days of the week experienced higher disruption rates.
- Weather-related and carrier-related factors were among the leading causes of cancellations.

## Skills Demonstrated
- Data Cleaning and Transformation
- Exploratory Data Analysis

## Conclusion
This project showcases the use of Power BI to analyze large-scale aviation data and communicate actionable insights through an intuitive and interactive dashboard.
